# Changelog

本项目采用语义化版本号。当前版本是前端演示版，生产后端能力另行发布。

## [0.1.0] - 2026-07-13

### 新增

- 首页、创作、全部模板和作品库组成的完整前端信息架构。
- 模板优先创作流程，支持竖屏样片预览、模板选择和独立全部模板页面。
- 商品图与人像模板的素材选择、授权提示、积分预估和模拟生成流程。
- 作品完成日期、输出规格、来源素材、文件信息、保存期限和积分记录。
- 作品重命名、删除二次确认、下载和再次制作操作。
- 登录头像菜单、模拟扫码登录、积分充值、积分流水和上传素材管理。
- 桌面端和移动端响应式导航与页面布局。

### 优化

- 隐藏非必要高级设置，减少内部术语和演示型提示语。
- 为任务结算、失败退款和支付到账增加防重复处理。
- 将任务状态构造提取为可测试的纯函数。
- 增加本地创作草稿结构校验，旧数据或损坏数据可安全回退。
- 增加日期格式、状态转换、作品操作和 API Client 测试。

### 已知限制

- 登录、支付、上传和 AI 生成均为浏览器内模拟。
- 页面刷新后，仅创作草稿会保留；账号、任务、积分和作品状态会恢复演示种子数据。
- `http` API 模式尚未覆盖全部写操作，不能作为生产业务后端使用。
- 示例图片和视频仅用于产品体验，正式商用前需要完成素材授权复核。
- 暂未提供团队协作、作品公开分享、社交平台发布和国际化。
